CTF 用于伪造ip的http请求头
Post：2023-05-13 15:09:47 Tags：/ ctf / Visit：

X-Forwarded-For:

该字段包含客户端通过代理服务器转发的IP地址，这通常用于验证客户端是否通过代理服务器进行通信。

Client-ip:

该字段包含客户端的IP地址，用于验证客户端是否与服务器在同一网络中。

X-Client-IP:

该字段包含客户端的IP地址，用于验证客户端是否与服务器在同一网络中。

X-Remote-IP:

该字段包含服务器返回给客户端的IP地址，用于验证服务器是否与客户端在同一网络中。

X-Rriginating-IP:

该字段包含服务器缓存中保存的IP地址，用于验证服务器是否在本地缓存中保存了客户端的IP地址。

X-Remote-addr:

该字段包含服务器返回给客户端的IP地址，用于验证服务器是否与客户端在同一网络中。

HTTP_CLIENT_IP:

该字段包含客户端发送HTTP请求时使用的IP地址，用于验证HTTP请求是否来自同一客户端。

X-Real-IP:

该字段包含客户端的真实IP地址，用于验证客户端是否与服务器在同一网络中。

X-Originating-IP:

该字段包含服务器返回给客户端的IP地址，用于验证服务器是否发起了该HTTP请求，该地址应与HTTP响应相匹配。

via:

该字段包含通过代理服务器转发的 IP 地址，这通常用于验证客户端是否通过代理服务器进行通信。
